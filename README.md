# 🚀 MINI_PROJECT

A collection of beginner-friendly projects built using web technologies and programming concepts.

---

## 📌 Projects Included

### 🕒 Analog Clock

* A real-time analog clock built using HTML, CSS, and JavaScript
* Shows current time with hour, minute, and second hands

### 🧮 Calculator

* A simple calculator for basic arithmetic operations
* Supports addition, subtraction, multiplication, and division

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

## 🚀 How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/MINI_PROJECT.git
```

2. Open any project folder:

```
cd MINI_PROJECT/Analog_Clock
```

3. Run the project:

* Open `index.html` in your browser
